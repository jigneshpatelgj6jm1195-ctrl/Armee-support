ArMee Technology Services Pvt. Ltd.
School Complaint Management Manual
Step 1: Select Project
Select GK (Gyan Kunj) or ICT Project.
Step 2: Enter DISE Code
Type the school's 11-digit DISE Code. School details will auto-load.
Step 3: Verify School Info
Verify the Principal Name and Contact Number. Click ✏️ Edit if updates are needed.
Step 4: Select Details
Choose the Equipment and Nature of Complaint from the dropdown lists.
Step 5: Enter Quantity
Specify the quantity. If Qty > 1, the app automatically guides you through each item sequentially.
Step 6: Serial Number & Scan
For each item, type the Serial Number or tap 📷 Scan to scan the barcode.
Step 7: Capture Photo (Mandatory 📸)
Camera/Gallery photo is mandatory for each complaint item. Submission is blocked without a photo.
Step 8: GPS Location (Mandatory 📡)
Allow location permissions. GPS coordinates will be captured automatically during submission.
Step 9: Submit Complaint
Tap Submit. The system generates a Case ID (Ticket Number) and logs it.
Step 10: Completion
Select to register another, share a text summary via WhatsApp/Email, or exit.
